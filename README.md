## App Context
**I am tasked to develop** a mini e-commerce web application for my capstone 3 in our full-stack web development bootcamp. The app has a catching landing page which also features a view of the products. A shop page where products are showcased, and when viewed, it directs users to the full view of the product where they can add to card or checkout. The app also features user management of roles, a normal user can only view and buy products, whereas an admin has an access to product management page.
